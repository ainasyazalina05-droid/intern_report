Tasks performed, featured projects, and primary technical responsibilities throughout the industrial training period are presented.

Tasks Performed

Industrial training was primarily focused on mobile application development using Flutter. Responsibilities included developing user interface components, integrating APIs, testing application features, and assisting with internal training-related tasks throughout the internship period.

Development Workspace Flutter-based application development, testing, and team collaboration workflow.

UI Development
Interface components and application layouts were developed and refined based on project requirements and design feedback to improve usability and user experience.

API Integration
API integration was implemented to connect the application with backend services, while API responses and displayed data were tested to ensure proper functionality and accurate data presentation within the application.

Testing and Bug Fixing
Application features were tested systematically to identify bugs and unexpected behavior. Minor UI and functionality issues were resolved, while more complex issues were reported to the development team for further review.

Training & Version Control
Modules and presentation slides for Flutter training sessions were prepared as part of internal training support. Git and GitHub were also used daily for version control, task management, and collaboration with team members.

External Tools & Services
Basic exposure was also gained in using Sentry for error monitoring and OneSignal for push notification integration within mobile applications.

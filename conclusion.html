Conclusion

Reflections and recommendations from the training period.

End of Industrial Training

Twenty weeks of learning, working, and improving are summarized here through the final reflection and suggestions.

Suggestions for the Organization / Polytechnic / Students

For Company

  • Provide clearer task instructions
  • Improve documentation for interns

For Polytechnic

  • Include more practical projects
  • Emphasize modern frameworks like Flutter

For Students

  • Practice coding regularly
  • Improve communication skills
  • Be proactive in learning
Overall Conclusion

The 20-week internship provided valuable exposure to a real working environment in mobile application development. During this period, involvement was mainly in UI development, API integration, testing, and assisting in training activities.

This experience helped improve technical skills such as Flutter development and API usage, as well as soft skills including communication, time management, and teamwork. Time and daily tasks were also managed more systematically in a real work environment.

Although there are still areas for improvement, this internship has provided a strong foundation for future work in the IT industry.
